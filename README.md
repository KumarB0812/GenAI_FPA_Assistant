# GenAI_FPA_Assistant
This is a Generative AI Assistant that will take audit reports, financial documents etc. for exploration and Summarization. Gives clean report summary, extract text, draft report, recognize patterns and takes use cases to produce output.

Problem: Manual reporting is time consuming and nobody wanna do that, seriously.

  
Solution: AI powered summarization and report generation.(because, why not)
  
Features:
  1. Summarization
  2. Risk extraction
  3. Report drafting
  4. Use-case prioritization
  5. Governance check


Impact:
  1. Reduces manual effort(for sure)
  2. Improves consistency(hack yeah)
  3. Enables faster decision-making(so you can wrap up quickly and go home, jk)


Technical info:
1. Streamlit - for the deployment
2. Groq - llama-3.1-8b-instant LLM API has been used, however any LLM model can be used or more than one models can be used.
3. Python - Logic, API call etc.
4. Git - Version control, track changes to code, feature update etc.
